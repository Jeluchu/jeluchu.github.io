App promo

  • Sobre el proyecto
  • En el mes de Junio, el cliente pidió la renovación de su web al completo, además de la realización de una pequeña aplicación móvil en la que se pudieran realizar comentarios en sus publicaciones, opción de leer las entradas más tarde, y de poder compartir las mismas. También se requería que tanto la web como la app móvil pudiera recibir las notificaciones push que se enviaban desde un panel de control.
  • Informe final
  • La web se realizó en los tres meses de verano, y se realizó la inauguración el 1 de septiembre de 2018, en la cuál se mantendrá un seguimiento durante un tiempo para evitar problemas o realizar las mejoras necesarias. La aplicación se publicó en la Play Store para la descarga en dispositivos android de forma totalmente gratuita, y que poco a poco recibirá futuras actualizaciones con mejoras.
Características del proyecto

La duración del proyecto ha sido de 3 meses, con la creación de una web que use la tecnología de Wordpress para que el cliente tenga un panel de control sencillo de usar. La propia aplicación mediante un plugin recoge todos los datos y muestra de ese modo las entradas dentro de la misma, usando el lenguaje Java.
