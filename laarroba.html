La Arroba

Una nueva forma de comprar

Optimización e impulso del pequeño comercio en los barrios y distritos de Madrid. Se trata de un directorio comercial, para que cualquier persona pueda acceder al buscador y encuentre cualquier tipo de comercio o servicio dentro del barrio, buscando o bien por lo que desea, o por las diversas categorías o servicios que se ofrecen. Este servicio está disponible tanto en versión web como en versión móvil.

VISITAR SITIO WEB
Directorio

Dentro de este proyecto se ha realizado una página web para informar de las diversas iniciativas que realiza dicha asociación de comerciantes, además de disponer de un directorio comercial para localizar los comercios cercanos de la zona.

Página Web
App Móvil

Para ir más allá en el fomento del consumo de los pequeños comercios se quiere realizar una aplicación para dispositivos móviles en los cuales las personas puedan ver las diversas ofertas de cada comercio además de acumular puntos para posibles descuentos. Actualmente la app está en una fase alpha, se puede testear desde la Play Store pero aún está en una primera fase.
